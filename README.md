# Next Toppers Book and T-Shirt Inventory Management

A responsive web application for managing Next Toppers book inventory and T-shirt inventory.

## Current project status

The repository has been created and the approved requirements baseline has been recorded. Application development has not started yet.

## Main modules

1. Book Inventory
2. Book Allocation and Return
3. Book Photo Recognition
4. T-shirt Stock Inventory
5. T-shirt Allocation
6. Annual Free T-shirt Entitlement
7. Paid T-shirt Issue with HR Approval
8. User-wise and Logo-wise History
9. Excel and PDF Reports
10. Admin Management

## Quality requirements

- Automated testing
- GitHub Actions CI/CD
- Responsive design for mobile, tablet and desktop
- Support for Android, iPhone, iPad, Windows, macOS and Safari
- Cross-browser optimization

See [docs/PROJECT_REQUIREMENTS.md](docs/PROJECT_REQUIREMENTS.md) for the full locked requirements.
